
This describes modifications / additions to the current repository that are
used by IGLib libraries and derivatives.

The branch IGLib/21_11_21_IGLibInfoAdditions is the base branch for such 
additions, and all other branches with IGLib additions shoulld merge this
branch in some way in orderr to add the current text file and its containing
directory. Presence of this directory is immediate indication that you are
dealing with a branch or tag that is used for IGLib.
